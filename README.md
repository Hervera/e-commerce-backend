# E-Commerce Frontend

A modern and responsive e-commerce frontend application built with **Angular 19.1.5**. This app delivers a smooth shopping experience, enabling users to browse products, manage their cart, and proceed through checkout with an intuitive user interface.

## Features

- Product listing and product detail views  
- Shopping cart management (add, update, remove items)  
- Responsive design for desktop and mobile  
- Reusable Angular components and services  
- Clean and scalable application architecture  

## Tech Stack

- **Framework:** Angular 19.1.5  
- **Language:** TypeScript  
- **Styling:** CSS / SCSS  
- **State Management:** Angular Services  
- **Build Tool:** Angular CLI  

## Prerequisites

Ensure you have the following installed:

- **Node.js** (v18 or later recommended)  
- **npm** or **yarn**  
- **Angular CLI** compatible with Angular 19
